2. END-TO-END ARCHITECTURE FLOW VERIFICATION
--------------------------------------------------------------------------------

Requested flow:
  SDL -> Strategy Builder -> Indicator Registry -> Smart Money Registry
       -> Strategy Compiler -> Strategy Model -> Execution Pipeline (arch only)

Verified actual flow (live-tested against moving_average_cross.yaml):

  1. app.sdl.StrategyParser.parse_file()        -> raw dict
  2. app.sdl.StrategyValidator.validate()        -> StrategyDefinition
  3. app.strategy_builder.StrategyContext(...)   -> bundles SDL doc +
                                                     IndicatorRegistry +
                                                     SMCRegistry
  4. app.strategy_builder.resolve_components()   -> classifies every SDL
                                                     IndicatorSpec against
                                                     IndicatorRegistry first,
                                                     then SMCRegistry
  5. app.strategy_builder.StrategyValidator      -> missing/ambiguous/
                                                     duplicate/circular/
                                                     invalid-reference/
                                                     version checks
  6. app.strategy_builder.StrategyCompiler       -> DependencyGraph +
                                                     ExecutionPipeline +
                                                     SHA-256 checksum
  7. app.strategy_builder.StrategyModel          -> immutable, hashable,
                                                     serializable, versioned
                                                     output (VERIFIED)
  8. Execution Pipeline                          -> description-only
                                                     (ExecutionStep list);
                                                     NO execution code exists
                                                     anywhere in the codebase

This flow was executed live during this audit (not just unit-tested) and
produced correct, deterministic results. See StrategyBuilderAuditReport.txt.
